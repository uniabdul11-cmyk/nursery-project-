# Paradise Nursery Shopping Cart (React)

A simple e-commerce shopping cart app for the **Paradise Nursery** online plant shop.

## Features
- Browse plants grouped into categories
- View plant thumbnails, names, descriptions, and prices
- Add items to cart (button disables after adding)
- Update quantities, remove items, and see totals
- Navigation bar with dynamic cart count

## Tech
- React + Vite
- Redux Toolkit (cart state)
- React Router (page navigation)

## Run locally
```bash
npm install
npm run dev
```
